# Agents Demystified

To install dependencies:

```bash
bun install
```

To run:

```bash
export ANTHROPIC_API_KEY=your-anthropic-api-key
bun run step1.ts
```

This project was created using `bun init` in bun v1.2.13.

[Bun](https://bun.sh) is a fast all-in-one JavaScript runtime.
